# 👥 Client Management App

Track clients, projects, payment status, and deadlines all in one place — designed for freelancers.

## ✨ Features
- Add/edit/delete clients
- Track project status
- Simple dashboard UI

## 🛠️ Built With
- HTML, CSS, JavaScript
- LocalStorage (no backend)
